# 拼多多自动上架工具

当前已支持 ERP 账号密码登录：价格册查询会优先复用 `erp.storage_state`，如果登录态失效或被踢回登录页，会自动使用 `erp.username` / `erp.password` 登录，登录成功后保存新的 `storage_state`，再继续读取价格册价格。

## 桌面双击启动

运行 `create_desktop_shortcut.ps1` 会在当前用户桌面创建 `拼多多自动上架` 快捷方式。以后双击这个快捷方式，会先检查本地工作台是否已经启动；如果已启动就直接打开 `http://127.0.0.1:8765/`，如果未启动就运行 `desktop_tool.py` 并等待工作台可访问后自动打开网页。

## ERP 优质价规则

自动上架时使用“订单 > 价格册 > 优质价”里的价格作为基础价，再乘以商品 `meta.yaml` 里的 `price_multiplier` 填到平台售价。

尺寸图文件名要用于精确匹配优质价：

- 文件名里包含完整型号，例如 `8065-25古铜色.jpg`。
- 程序会先搜索基础型号 `8065`。
- 如果基础型号没有返回结果，会再用完整型号重试，例如 `8065-25`。
- 搜索结果里会精确匹配 `8065-25`，不会误拿 `8065-20` 或 `8065-30`。
- 如果同一型号有多个颜色，会继续按文件名里的颜色匹配，例如 `古铜色`。

## 配置 ERP

复制 `config.example.yaml` 为 `config.yaml`，填好 ERP 地址和页面选择器。账号密码建议用环境变量：

```powershell
$env:ERP_USERNAME="你的ERP账号"
$env:ERP_PASSWORD="你的ERP密码"
```

`config.yaml` 中保持：

```yaml
erp:
  username: "${ERP_USERNAME}"
  password: "${ERP_PASSWORD}"
```

需要用 F12 填好的选择器：

- `username_input`: ERP 登录页账号输入框
- `password_input`: ERP 登录页密码输入框
- `login_submit`: ERP 登录按钮
- `search_input`: 价格册搜索框
- `search_submit`: 搜索按钮，可留空，留空会按 Enter
- `price_cell`: 价格显示元素
- `login_error`: 登录失败提示，可留空
- `wait_timeout_ms`: ERP 页面和价格接口等待时间，建议保持 `30000`，避免优质价接口偶发慢响应时过早失败

真实账号、密码、`states/*.json` 不要提交到 git。

## 图片空间模式

当前工作台优先按拼多多后台“图片空间路径”生成上架包，不再要求每个商品都准备本地商品文件夹。

正常使用时填写：

- `价格倍数`：例如 `1.8`，用于把 ERP 优质价换算成平台拼单价。
- `价格尾数`：选择 `8` 或 `9`；程序会先四舍五入到两位小数，再把小数点后第 2 位调整为所选尾数。
- `材质`：例如 `黄铜`、`锌合金`、`铝合金`。
- `图片空间路径`：例如 `2026/8105-8255`。

图片空间目标文件夹下仍需包含 3 个子文件夹：

- `主图`
- `详情页`
- `尺寸图`

程序会用图片空间“尺寸图”的文件名生成 SKU，并按文件名匹配 ERP 优质价。本地商品文件夹现在只作为兼容旧流程的可选输入：如果该目录存在 `meta.yaml`，会继续读取其中的类目、材质、库存等默认值；如果不存在，也不会阻止生成上架包。

品牌不在本地模板里写死。不同店铺对应不同品牌时，发布页选完类目后由拼多多后台显示该店铺可选品牌；工作台不会默认填写“梵居匠”，避免跨店铺选错品牌。

尺寸图文件名里的 `单孔`、`吊坠` 等安装/规格描述不会拼进 ERP 价格册型号。例如 `2715云栖-铬色-单孔` 会用 `2715` 查询价格册，并用 `铬色` 匹配价格册颜色 `铬`；如果价格册候选里有唯一的 `2715-单孔` 或 `2715-吊坠`，会自动匹配这个安装后缀，不会误拿 `2715-25直径`。如果安装描述里带括号尺寸，例如 `6601-古铜色-单孔（24）`，括号数字会按图片标注尺寸处理，不作为优质价型号；如果旧缓存已经生成了 `6601-24` 这类查询，价格册没有该型号但有唯一的 `6601-单孔` 或 `6601-吊坠` 时，也会回退到对应安装后缀。部分图片命名颜色会按 ERP 价格册颜色归一：`钛银` 对应系统 `亮镍`，`亮金` 对应系统 `玫瑰金`，`PVD金` 对应系统 `铬PVD`，`铬色` 对应系统 `铬`。如果尺寸图文件名写的是 `亮镍`，拼多多规格名会显示为 `钛银`，但 ERP 查价和规格编码仍使用系统颜色 `亮镍`。

尺寸图 SKU 会按型号后的尺寸数字升序创建，例如 `96`、`128`、`192`，避免按文件名字典序排成 `128`、`192`、`96`。点击“打开发布页开始填充”或“填规格价格测试”时，发布页商品级“商品编码”会自动填写工作台里的价格倍数，例如 `1.6`；满件折扣默认填写 `9.9` 折。

部分店铺进入发布流程后，会先出现“商品主图 / 商品标题 / 商品分类 / 下一步，完善商品信息”的发布前信息页。工作台会先识别这个页面：如果主图已存在，会直接复用；如果主图为空，会尝试用上架包主图上传；随后填写标题、选择推荐分类并点击下一步，再继续进入完整商品信息页填属性、规格和价格。
